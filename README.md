# 青盾成长助手 App 原型

这是“青盾计划”家长端 App 的第一版可运行网页原型，定位为：

> 面向家长的青少年健康用网建议与协同管理 App。

## 打开方式

直接双击 `index.html`，或用浏览器打开：

`C:\Users\ASUS\Documents\try\qingdun-parent-app\index.html`

## 当前功能闭环

- 家长首页：查看孩子数字驾照等级、综合分、风险等级、本周建议。
- 数字驾照测评：通过 8 道情境题刷新五维能力分数。
- 用网风险报告：生成总分、风险等级、短板维度和优先建议。
- 家庭用网契约：勾选本周执行规则，并保存复盘。
- 家长建议中心：按短视频、网络欺凌、隐私暴露、亲子冲突等场景提供沟通话术。
- 孩子档案：可编辑昵称、年龄和年级。
- 本地保存：演示数据保存在浏览器本地，不上传服务器。

## 演示路线

1. 打开首页，介绍孩子当前风险与数字驾照等级。
2. 点击“开始测评”，完成 8 道题。
3. 系统跳转到风险报告，展示分数变化和短板维度。
4. 进入“契约”，勾选家庭规则并保存复盘。
5. 进入“建议”，展示家长面对具体场景时可以怎么沟通。

## 第一版边界

本原型不读取真实聊天记录、不采集完整上网记录、不定位、不做人脸识别、不进行真实锁机或平台封禁。

这些限制是故意保留的，因为项目面向未成年人，比赛阶段应优先展示“建议、协同、测评、契约”的产品闭环，避免过度监控和隐私合规风险。
